# 小红书内容大纲

基于选定的选题，生成 9 个不同风格的内容大纲，并给出综合评分和推荐。

## 使用方法

```
/xiaohongshu-outline
```

## 功能

- 调用 prompt-generator 生成采访提示词
- 采访用户（6个问题）
- 生成 9 个版本的内容大纲（3×3矩阵：结构×受众）
- 提供综合评分和推荐版本

## 前置要求

- 需要先完成选题策划（`/xiaohongshu-ideation`）
