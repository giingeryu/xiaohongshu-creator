# 小红书选题策划

基于对标分析和账号定位，生成精准的选题方案。

## 使用方法

```
/xiaohongshu-ideation
```

## 功能

- 读取对标分析文档和账号配置
- 询问用户想创作的主题
- 生成 3-5 个选题方向
- 每个选题包含：标题方向、目标人群、核心价值点、爆款潜力评分

## 前置要求

- 需要先完成账号配置（`/xiaohongshu-setup`）
- 建议完成对标分析（`/xiaohongshu-analysis`）
