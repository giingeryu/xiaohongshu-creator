# Prompt Generator

专业的 Prompt 生成器，基于 meta-prompting 最佳实践生成高质量 prompts。

## 核心特性

- 任务分解：将复杂请求分解为更小的子任务
- 独立验证：使用不同的"专家"进行独立审查
- 迭代验证：强调验证过程，减少错误和幻觉
- 避免猜测：不确定时明确声明
- 专家协作：召唤专门的"专家"角色协助

## 使用方法

`/prompt-generator` - 启动交互式生成
`/prompt-generator <需求>` - 直接生成
