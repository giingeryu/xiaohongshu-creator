# 小红书笔记创作助手

专业的小红书内容策划工具，提供从选题到发布的全流程创作支持。

## 功能特性

- **创作者配置**：智能分析或手动填写账号定位
- **爆款笔记拆解**：使用小红书点点分析爆款笔记的结构、标题、钩子等
- **推荐类似笔记**：基于已分析笔记推荐相似爆款内容
- **生成对标分析文档**：自动生成包含综合分析和选题建议的完整报告

## 使用方法

### 安装

将此 skill 复制到你的 Claude Code skills 目录：

```bash
cp -r xiaohongshu-creator ~/.claude/skills/
```

### 快速开始

在 Claude Code 中使用：

```
/xiaohongshu-creator
```

## 工作流程

1. **配置账号定位** - 使用小红书点点分析你的账号
2. **分析爆款笔记** - 使用提供的提示词在点点中分析对标笔记
3. **推荐类似笔记** - 让点点推荐更多相似爆款（可选）
4. **导出分析文档** - 生成包含综合分析的完整报告
