# 点点平台提示词生成器

专门为点点平台设计的提示词生成器，基于 meta-prompting 最佳实践，生成500字以内的精简高效提示词。

## 核心特性

- **字数控制**：严格控制在500字以内
- **精准聚焦**：去除冗余，保留核心指令
- **结构清晰**：使用简洁的分段和标记
- **可执行性**：确保提示词直接可用
- **验证机制**：内置质量检查

## 使用方法

`/diandian-prompt` - 启动交互式生成
`/diandian-prompt <需求>` - 直接生成

## 适用场景

- 小红书内容搜索
- 对标分析任务
- 内容创作指导
- 数据筛选策略
- 快速执行任务
